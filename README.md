# Systematic – Accounting Firm Website

A clean and responsive static website built using **HTML5** and **CSS3**.  
This project showcases a fictional accounting firm called **Systematic**, featuring sections such as services, testimonials, blog posts, contact form, and footer information.

This website is ideal for beginners practicing front-end development and layout structuring using pure HTML and CSS.

---

## 🖼️ Features

- Fully static website (HTML + CSS only)
- Clean and modern layout
- Navigation menu with hover effects
- Service section with icons
- Testimonial/Quote section
- Contact form UI (front-end only)
- Organized folder structure (CSS + images)
- Font Awesome icons included
